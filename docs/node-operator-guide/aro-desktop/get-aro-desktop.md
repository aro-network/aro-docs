# Get ARO Desktop

## About ARO Desktop

**ARO Desktop** is a lightweight desktop application that delivers a smooth, user-friendly, and highly stable experience for users of all levels.

It is currently available for **macOS**, **Windows**, and **Linux**. To confirm whether your specific OS version is supported, please visit the **ARO Dashboard -> Add a New Node** page.

## How to Install ARO Desktop

Simply download and install an ARO Desktop node directly from the **ARO Dashboard -> Add a New Node** page. Follow the on-screen instructions to complete the setup.

For detailed installation steps and additional guidance, refer to:

- [ARO Desktop Installation Guide](./aro-desktop-installation-guide)
- [FAQ & Troubleshooting](./aro-desktop-faq-troubleshooting)

# Get ARO Mobile

## About ARO Mobile

**ARO Mobile** is a lightweight mobile application that lets you run your ARO node and earn with it wherever you are.

It is currently available for **Android OS**. To confirm whether your specific OS version is supported, please visit the **ARO Dashboard -> Add a New Node** page.

## How to Install ARO Mobile

Simply download and install an ARO Mobile node directly from the **ARO Dashboard -> Add a New Node** page. Follow the on-screen instructions to complete the setup.

For detailed installation steps and additional guidance, refer to:

- [ARO Mobile Installation Guide](./aro-mobile-installation-guide)
- [FAQ & Troubleshooting](./aro-mobile-faq-troubleshooting)

# Get ARO Server

## About ARO Server

**ARO Server** is a dedicated ARO node type for server or workstation environments, delivered as a software image.

You can choose how to run it based on your own infrastructure preferences:

- Bare-metal deployment
- Virtual machine (VM) deployment

ARO Server is designed for **pro-level users** who need advanced control, tuning, or specialized deployment patterns. If you do not have specific advanced requirements, other node types may be a better fit.

## ARO Server and the Former ARO Client

ARO Server is the continuation of the former **ARO Client**.

We renamed this node type to **ARO Server** for two reasons:

1. Its capabilities and deployment form have evolved.
2. The term "client" is generic and may cause user confusion.

If you have already installed the previous ARO Client and it is still running normally, there is no need to worry. Your node remains compatible with the current unified incentive rules.

## How to Install ARO Server

Download and set up an ARO Server node from **ARO Dashboard -> Add a New Node**, then follow the installation flow based on your environment.
